# base-flask-api

Fire up a new api

## Quick start

`$ git clone https://github.com/*/*.git`



**Create and activate a new conda environment with the appropriate modules**

`$ conda env create -f environment.yml`


**Create a new database table**

`$ python -m app.db.db_create_new`

**Run application**

`$ python -m run`
